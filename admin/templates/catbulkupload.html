Note : To Upload The Category TSV File
  1. Click on the Sample images TSV file to download the zip
  2. Extract the zip file.
  3. There are two tsv files in the zip archive.
  4. Don't Modify Or Delete The First Line Of Category TSV File
  5. Modify The Dummy Category Details From The Second Line Of Category TSV File
  6. Append The Category Details In The Given Format
  7. Enter the Category Ids Correctly Given in the Category Details TSV File
  8. Enter the Type of Category Correctly as Parent or Child (lower case only) in the Category TSV File
  9. Enter the Category_Parent_Id as Zero(0) for Main Category in the Category TSV File
