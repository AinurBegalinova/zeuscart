Header and Custom Header

ZeusCart 2.3 is a sophisticated, user friendly shopping cart program used all over the world to build and manage an Internet store. Our ecommerce software is one of the very best shopping cart programs available today, and it contains all the tools you need to run a profitable online store.

A simple truth: people don't buy from unprofessional-looking stores. So don't settle for an ecommerce system that does not give you full control on the look and feel of your online storefront. With ZeusCart 2.3, you can build a nice looking, powerful ecommerce store, seamlessly integrated with the rest of your Web site.

The header is at the very top of every web page. It usually contains a logo for website and some links to connect with the other pages of site and provides links to give entire information about the site.

 

 

The included sections in header are

  • Welcome Message and Time Zone: The welcome message will display as "Guest", when the user has not yet registered and will display with the "Display Name", when the user has logged in with their registered name. The date and time will be displayed based on the time zone selected by Admin.
  • Logo & Google Ad: ZeusCart 2.3 provides enough space to display your company logo image in site. The logo uploaded through Admin panel will be displayed in the site. The logo added will be displayed in 223 x 70 dimensions. Based on the added Google AdSense Code added by Admin, the Google Ads will be displayed near the Logo. So Click on the Ad you will be directed to the corresponding Ad's site.
  • My Account: When a user had registered and wants to view the personal profile, click on "My Account" link will direct him to "Account Dashboard" page and from there ZeusCart 2.3 provides facility to update the personal information already entered and do any of actions like viewing Orders, Product Reviews, Wish List and much more.
  • My Wishlist: User can view the wishlist page directly from home page and user has provided benefit to add any number of products in their wishlist. Click on "My Wishlist" link will direct you to the "My Wishlist" page and from there you can check out your favorite product.
  • My Cart: The products that are added in "Your Shopping Cart" can be viewed when user clicks on "My Cart" link. User can check out a product directly from the shopping cart. In ZeusCart 2.3 even a Guest user can check out a product directly just by entering the mail id. When a Guest user checkouts a product then automatically he will become a member of ZeusCart 2.3.
  • Checkout: Click on this link you can checkout for a product. Using ZeusCart 2.3 you can checkout a product as a Guest User or as a Registered User.
  • Login: After your registration, click on this link will direct you to the "login page" to enter your "E-mail Address" and "Password" to get in to your own personalized page in the site. The email address will be the mail id given while your registration.
  • Header Menu: The special feature of ZeusCart 2.3 is to list the web pages added by Admin. It helps Admin to create the web pages by own. This is the place where all the web pages created by Admin to display it in site. User can click on the link displayed in this area will direct him to the web page created by Admin. The header menu has the link for Home to get in to Home page from anywhere in site. This is a default and cannot be changed and after that only the created web pages will be displayed. Click on More to view all the other web page names created by Admin.
  • Search: The search panel helps in finding out of products based on categories by just entering the keywords for searching product. Enter the keyword of product to be searched in the text box and select the category name displayed in list box and enter "GO" button will display the list of products related to entered keyword and selected category name.
  • Start here: If the user needs to register with site, click on "Start here" link will direct to the Registration page to enter information of user.

 

Custom Header Content

User can view the content added by Admin using "Custom Header" module in Admin Panel. The added content will be displayed in every pages of site. Custom Header Content helps user to view the announcements, or any offers or etc added by Admin.
