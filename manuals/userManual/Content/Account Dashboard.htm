Account Dashboard

ZeusCart 2.3 welcomes user with attractive Account Dashboard every time they login to their account. Account Dashboard acts like statistics of each and every activities. It gives the information of recent order made by you in a grid. The order details will be Order number, Order date, name of user to whom the product is sent, order amount and the status of order. It displays the account information like email id and link to change the password for your account. It shows the details newsletter subscription whether you have subscribed for newsletter or not.

 

  1. Login to your account using Email Address and Password. Then you will be getting in to Account Dashboard page.

a

 

  1. Recent Orders: The list of order details will be displayed in a grid with the information of Order number, Order date, name of user to whom the product is sent, order amount and the status of order.
  2. Account Information
    • Contact Information: Displays the information of Display Name and Email Address added while registration.
      • Click Change Password link will get you to Edit Account Information page.
      • Click Edit link will also get you to Edit Account Information page.
    • Newsletter Subscription: Displays the information that user has subscribed for Newsletter or not.
      • Click Edit will get you to Newsletter Subscription page.
