Web Page Content

The Web Page Content is considered to be body of web page. Here the web page content is the center section of home page. It includes two different sections like Home Page Banner and Featured Products.

 

 

  • Home Page Banner: The banner image will be displayed in the top of home page. The image will be added by Admin and he will link the banner with any URL. So when user clicks on the banner image, he will be directed to the URL given by Admin. So this will act as an advertisement for site.
  • Our Featured Products: The products displayed are considered as the featured products in the site. The featured products will be selected by Admin which will be displayed here. You can view product image, name, ratings added and Price. From here, User can add the products to wishlist. The featured products will be displayed in Home page and Main category landing page.
    • Click "Add to Wishlist" link to add the corresponding product in to your own wishlist page. Only registered user can add the products to Wishlist page.
    • Click More... link will get you to "All Featured Products" page where you can view all the featured products and in that page we have provided paging to view all featured products. Click on the page numbers to go to corresponding pages and click and to go back and to go next to the page where you are now. By default Eight products will be listed in a page.
  • Best Selling Products: The products will be displayed based on the number of fast moving products in market. The number of orders that are made for the product and also the number of high quantity which has been moved for that product. By default Four products will be listed in this section.
