Address Book

ZeusCart 2.3 provides you the new feature of creating your own personal address book. You can save number of address details using this address book. The created can be used while you checkout for a product. For ease use of user, we have added this feature. So while checkout you can just select the address you wish to use for billing and shipping of product.

 

  1. Login to your account using Email Address and Password. Then you will be getting in to Account Dashboard page.
  2. Click "Address Book" link available in the left column on My Account section will open Address Book page.
  3. Click Add Contact link

 

 

  • Group Name: You can provide the name for the group for which you wish to add this contact.
  • First Name: Enter the first name of contact.
  • Last Name: Enter the last name of contact.
  • Company: Enter the company name for contact.
  • Email Address: Enter the valid email address you wish to add. Example: name@domainname.com
  • Address: Enter the complete address for contact.
  • City: Enter the name of city.
  • Sub Urb: Enter the name of sub urb.
  • State/Province: Enter the state name of contact.
  • Country: Select the country name from the list box.
  • Zip/Postal Code: Enter the postal or zip code for the contact.
  • Telephone: Enter the contact number.
  • Fax: Enter the fax number.

In the above fields Company, Email Address, Sub Urb, Telephone and Fax fields are optional.

  1. Click Create will display "Created!" message.
  2. Click View Contacts link will get you to Address Book page where you can view the details for created contact.

 

  1. Click on View Contact available inside the Address Book will display the complete information of that contact.

 

  • Click on Edit Address link will help you to update the contact details.
  • Click on Delete Address link will display "Are you sure to delete?" message.
    • Click OK will delete the contact details.
    • Click Cancel to exit form deletion.
