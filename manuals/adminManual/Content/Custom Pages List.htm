Custom Pages List

ZeusCart provides a special feature for Admin to create his own web page in site. If Admin knows HTML coding then he can write his own web page to get display in the site. This encourages Admin to get interest in the Web Page creation. To display your own web page, ZeusCart provides you separate place in footer to display the names of added web pages. So it will not get confused for user with other footer links like Privacy Policy, Terms & Services and so on. Admin has privilege to delete the web pages created, whenever necessary.

 

  1. Click CMS > Custom Pages List will open Custom Pages page.
  2. The list of all the web pages created currently will be displayed with the option to delete it.

 

 

Adding New Web Page

  1. Click button provided in the Custom Pages page will open Create Page page.

 

 

    • Page Name: Enter the name of web page to be created.

Example: sample.html, webpage.html

    • Page Title: Enter the title name for web page to be created.
    • Meta Content: Enter the description or content for webpage to be added.
    • Meta Keyword: Enter the related keyword to be used while searching for web page related to your web page.
    • Css Files: If you have added any style sheet script in HTML code, click "Browse" to upload the .css file.
    • If any Js Files: If you have added any java script, click "Browse" to upload the .js file.
    • Body Content: Enter the body content of HTML code to create your web page.
    • Click Create will add the Web Page name.

 

Updating the Status of Web Pages

  1. If Admin needs to make his created web page to get display in site, then go to Custom Pages page and select the Check Box available corresponding to your created web page. Then click Update Page Settings will display your web page name in the home page of site.

 

Deleting the Web Page

  1. Click link provided corresponding to the created web page name will display "Are you sure want to Delete this Page?" message.
    • Click OK will display "Page Deleted successfully" message.
    • Click Cancel will exit from deletion.
