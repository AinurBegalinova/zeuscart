Coupons

ZeusCart 2.3 supports the advanced marketing feature of providing coupons for specific products. It makes your users to have marketing with you efficiently. Admin has provided the facility of adding coupons to some specific categories in his own wish. ZeusCart 2.3 generates the code for coupon automatically. Admin has given privilege to send coupons to users who has made more transactions and to any one need. We help you select the users easily by using the search function.

 

  1. Click Promotions > Coupons will open Coupon Codes Management page.
  2. The list of coupons that are currently added will be displayed in a grid with the information of Coupon Code, Coupon name, Discount, Valid date, Status and link to send the coupon.

 

 

Creating New Coupon

  1. Click button will open Coupon Codes Management page.

 

 

    • Coupon Code: Automatically the Coupon code will be generated for ease of use. The generated coupon code will be displayed. We have provided facility for Admin to edit the coupon code whenever necessary.
    • Coupon Name: Suggest any name for your coupon and enter it in the available text box.
    • Discount: Enter the value of discount you are giving for that coupon and select whether the discount should be calculated in % (Percentage) or $ (USD) from the nearby list box.
    • Valid From: Select the starting date of coupon from which it is valid from the available scheduled calendar.
    • Valid To: Select the ending date of coupon until which it is valid through from the displaying scheduled calendar.
    • Minimum Purchase: You can enter the value of minimum purchase. The entered value will be taken as USD value i.e., Suppose if you enter 100$, then the user must buy a product with the minimum price of 100$ then only he will be qualified to use this coupon.
    • Number of Uses: You can enter the number of usage of this coupon. Suppose if you enter the value as 2 then the user can use the coupon only for 2 times for purchase.
    • Select Categories: The list of all Categories and its Sub Categories will be displayed with the Check Box. Select the Category or Sub Category by clicking the check box provided.

If you select a Category then it applied for its Sub Category too. If you select the Sub Category then it is applicable only for selected sub category not for its corresponding Category.

    • Click Submit will display "Coupon Code Generated Successfully" message.

 

Updating the Status of Coupon

  1. You can update the status by clicking the icons available in Status column of the grid in Coupon Codes Management page.
    • To activate a coupon, Click icon provided corresponding to the coupon, then it will display "Coupon Status Updated Successfully" message.
    • To deactivate a coupon, Click icon provided it will display "Coupon Status Updated Successfully" message.

 

Sending the Coupon to Users

  1. Click Send link available in the Send column, will open Coupon Codes Management page.
    • The list of all users will be displayed with the information of username, date of join, total purchase made  and total orders made in a grid.
    • Select Users: You can select the users to whom you wish to send the coupon by selecting the given criteria or you can send to all users too.
      1. Number Of Orders: Select the number of orders like Greater Than or Less Than or Equal To from the list box and enter the value in the displaying text box and then Click List will display the list of users who has made the orders based on your selected criteria.

If you select greater than and entered the value as 10 then the list of users who has made greater than 10 orders will be displayed.

      1. Total Purchase: Select the number of purchase like Greater Than or Less Than or Equal To from the list box and enter the value in the displaying text box and then Click List will display the list of users who has made the purchase based on your selected criteria.

If you select less than and entered the value as 5 then the list of users who has made less than 5 purchases will be displayed.

      1. Date of Join Between: Select the dates from the available scheduled calendar and click List will display the list of users who has joined in between your selected dates.

If you select the dates of 25.05.2008 and 25.06.2008 then the list of users who has joined in between the selected dates will be displayed.

      1. Click Send Coupon will send the coupons to users which you have selected based on the above any one criteria and will display "Coupon Code Send To Selected Users Successfully" message.

 

 

    • If you wish to send coupons to all users then all the Check Boxes available near the user and then click Send Coupon will send the coupon to all users and will display "Coupon Code Send To all Users Successfully" message

 

We have provided Paging for viewing the coupon details. Click on the page numbers to go to corresponding pages and click Prev and Next to go back and to go next to the page where you are now. By default Ten coupon details will be listed in a page.
