Category Bulk Upload

Using Category Bulk Upload module, Admin can upload unlimited number of Categories and Sub categories in a single click. Admin can upload the categories in TSV file (Tab Separated Value) format. Admin can download the sample file from ZeusCart 2.3 in Zip format. You can download three files named Category, Category details and read me. In categorydetails file, you can view the list of category names and its corresponding id values. category file is your main file where you should enter the category details. In the first line it consists of the fields names to enter the category details. You should not make any corrections/modifications in the field names. Next to the field names you can start entering your category details one by one. After finishing all your entries save the file and you just upload it in ZeusCart 2.3 then automatically all the categories will be added in ZeusCart 2.3. This Bulk Upload saves your time in adding category details step by step and do not need any internet facility while adding the category details.

 

  1. Click Catalog > Categories > Category Bulk Upload will open Category Bulk Upload page.
  2. Sample Category TSV File: Click icon will open a dialog box to download the sample Zip file.

 

 

    • After downloading Sample file from ZeusCart 2.3, you will be seeing the three files in Zip folder named category, categorydetails and readme.
    • In categorydetails file you can view the list of category names with the corresponding ids generated.
    • In category file you can view the field names such as category name, description, type and category parent id.
      • category name: Enter the name of your category.
      • description: Enter the description for the corresponding category.
      • type: Specify the category type either parent or child. Use only the lowercase to enter the type.
      • parent id: If you are entering the name of sub category, you can collect the parent category id from categorydetails file. If you are entering parent category name, you can specify as 0 in this field.
    • After completing the category entry, save the category file.
  1. Category TSV File: Click Browse button to upload the TSV file in which you have completed your all category entries.
  2. Click Submit will display "Category(s) Created Successfully" message.

Some important points to be remembered before bulk uploading the categories.

  • Before doing bulk upload of Categories, you must add one or two category names in online.
  • Instead of uploading the Categories and Sub Categories at a time, for ease use you can upload all the Category names first and then can add the sub category names. This will help you in getting the category ids easy.
  • If you add the sub category names the corresponding parent category should already be installed. You can not add the category and its corresponding sub category names in the same time.
  • You should not make any corrections in the field names and you can enter the category details based on field names provided.
