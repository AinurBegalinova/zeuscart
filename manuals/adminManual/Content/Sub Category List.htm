Sub Category List

This chapter explains you about how to view the sub category names added and edit or delete them. Using Sub Category List you can create a new sub category or even a main category. We have provided the functionality to search sub category names from the list.

 

  1. Click Catalog > Manage Categories will open Main Category List page.
  2. Click on the name of category will open Sub Category List page.
    • By default, the sub category names will be displayed based on the Category name you have clicked from the Main Category List page.

 

 

  1. Category name: Select the category name for which you need to view the sub category names.
    • Search: Enter the keyword that you know about a sub category name on respective text box provided below the corresponding field names and then click Search button will display list of sub category names based on keyword entered.
    • Click icon will open Edit Sub Category page.

 

 

      • Update the necessary information that you need to modify from the fields available.
      • If you wish to activate the Sub Category then click ON else click Off option.
      • Click Update Sub Category button will display "Sub Category <name of sub category> Updated Successfully" message.
    • Click icon will display "Are you sure want to Delete?" message.
      • Click OK will display "Sub Category <name of sub category> Deleted Successfully" message.
      • Click Cancel to exit from deletion.
  1. Click button will help you add the Category or Sub Category names in ADD CATEGORY page.
  2. Click View Main Category will get you back to Main Category List page.
