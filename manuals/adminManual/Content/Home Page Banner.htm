Home Page Banner

Admin can upload the banner image to be displayed in home page.

 

  1. Click Settings > Site Settings > Home Page Banner will open Home Page Banner page.

 

 

  1. Upload Banner: Click "Browse" to upload the image of banner. By default, the added banner image of any size will be resized in to 564x160 dimensions.
  2. Banner Url: Enter the name of URL to be connected while clicking home page banner.
  3. Click Set Banner button will display "Site Home Page Banner Updated Successfully" message.
