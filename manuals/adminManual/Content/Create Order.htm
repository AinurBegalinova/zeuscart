Create Order

Admin has the special feature of creating order from Admin Console. So if any request from customers to create any order by Admin, then Admin can create the order from Admin Console itself. This is an advanced feature added for easy use of Admin. Admin can enter the billing address and shipping address of customers and can create the order on behalf of them.

 

  • Click Orders > Create Order will open Create Order page.

 

 

  • Select Category
    • Category: Select the main category under which the ordered product is listed.
    • Sub Category: The list of sub Categories available based on the selected Category will be displayed. Select the sub category which you need to add.
    • Products: The list of products will be displayed based on the selected category and sub category names. Select the product for which you need to create the order.
    • Qty: Select the quantity of product you wish to order from the list box. When you select the quantity of product its unit price will be displayed near the Qty list box.
    • Click Add button will display the product details in the available grid below.
  • The grid consists of details like product name, quantity you have selected, unit price of product, corresponding shipping cost for product,and the total of product.
  • Click icon will display "Are you sure to delete" message.
    • Click OK will remove the product from grid.
    • Click Cancel to exit from deletion.
  • Address
    • Customer: The list of customers available will be displayed in the list box. Select the customer for whom you wish to create the order.
    • Paid Through: Select the mode of payment from the list box.
    • Billing Address:
      • Name: Enter the name in which the product bill should be delivered.
      • Company: Enter the company name if needed since it is an optional field.
      • Address: Enter the billing address to which the product should be sent.
      • City: Enter the name of your city.
      • SubUrb: Enter the name of your sub urban if it is applicable. Because it is an optional field.
      • State/Province: Enter the name of your state.
      • Country: Select the country name from the list box available.
      • Zip/Postal Code: Enter the Zip/Postal code of city.
    • Use Billing Address as Shipping Address: Click on this check box if shipping address is same as you have entered for your billing address else do not check the check box.
    • Shipping Address: If your shipping address is not same as billing address then follow the steps i to viii as like give in billing address.
  • Click Create Order button will display "Order Created Successfully" message.
