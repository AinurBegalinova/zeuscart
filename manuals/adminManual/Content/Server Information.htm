Server Information

Once you have installed ZeusCart 2.3 in your system, the information about your server will be displayed automatically in this page. This page provides you the complete details like PHP version, MYSQL details, server details, your system information and much more. Therefore, you can get all the details immediately from ZeusCart 2.3 itself.

 

  • Click Others > Server Information will open Server Information page.
  • The page will display the information of your server, PHP, MYSQL that is installed in the system.
