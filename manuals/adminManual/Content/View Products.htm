View Products

Using this module, Admin can view all the currently added products. This module helps Admin to add a product in site. The search engine supports Admin to search for product information fast and easy. Admin has privilege to edit/delete the product information, whenever necessary. Admin can add cross products for a particular product. Admin has the facility to add tier price and can attributes for the product.

 

  • Click Catalog > Products > View Products, will open Product List page.
    1. By default, the list of all products currently added in site will be displayed with Product Name, Brand name, model of product, MSRP, price and the options to Edit, Delete, to update the tier price, update special features for the product.

 

 

    1. Search: Admin can search the product using the known keywords.
      • The added Products will be listed with all its information like Name, Brand, Sku, Quantity, Msrp, Price, Status, Cse Enabled, Tag and Intro Date.
      • If you need to search for a product, enter the Keyword that you know in the text box provided below the corresponding field names.
      • Click Search button will display the list of products related to the provided keyword.

 

 

    1. Viewing Product Details - Click on the name of added product link will open PRODUCT DETAILS page.
      • If you need to update the product information click on button will open EDIT PRODUCTS page.
      • If you wish to add related products for the selected product, then click Add Cross Products To This Product link will open Add Cross Products page.
        • The Add Cross Products page displays the name of product that you have selected to add related products list.
        • Main Category: The list of all currently available categories will be displayed. Select the category name to add related product for the selected product.
        • Sub Category: Based on selected category name, the list of sub category names will be displayed. Select the sub category name to be added.
        • The list of products available under the selected category and sub category will be displayed with the information like Product Name, Brand, Model and MSRP.
        • Select the product that you need to add as related product by enabling the check box provided under the Select field in the grid.
        • Click Save button will display "Cross Products added Successfully" message.
    2. Adding a New Product - If you want to add a new Product, then click button.
      • When the Product is added, it will be added in a grid.
    1. Editing the Product Details
      • Click icon available in Product List page (Catalog > Products > View Product), will open Edit Products page.
      • The tabs that were available while adding the product will be displayed.
      • Modify all the necessary information that needs to be updated by clicking the respective tab.
      • Click on the Finish button available in Related Products tab will display "Product <name of the updated product> Updated Successfully" message.
    1. Deleting the Product
      • Click icon provided in Product List page (Catalog > Products > View Product), will display "Are you sure want to Delete this Product?" message.
        • Click OK will display "Product Deleted Successfully" message.
        • Click Cancel will skip from deletion.
