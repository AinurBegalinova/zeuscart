Admin Welcome Screen

Welcome to ZeusCart 2.3 Admin Console! The home page of ZeusCart 2.3 Admin Console seems to be attractive and valuable. The Statistics Page shows a range of useful information about the stores resource usage and order status. ZeusCart 2.3 has the advanced site statistics which gives you entire information about the site like Order, Sales, Products, Product Inventory and Customer Statistics. The main page of home page displays the information of Latest Orders and New Customers in a grid. The home page displays the link to view Help Manual and Logout link.

 

 

  • Site Statistics: It is considered as the friends for a Web Master. It figures what a webmaster reads, understands and analyzes. If the figures give out a good outcome, there are chances that he will relax. So ZeusCart provides you the complete information of site like number of customers, number of orders, number of Products and much more.
  • Admin Map: When you click on this link you can view the links to navigate through ZeusCart 2.3. Using the links provided you can get in to corresponding pages directly.
  • Store Front: Using this link you will be directed to store front. So this link helps you to view the changes immediately without entering the domain name in new window of the browser.
  • Help: Click on this link, will provide you the entire Admin Manual for your help.
  • Logout: Click on this link to exit from the site and will display "You Have been Successfully Logged out" message.
