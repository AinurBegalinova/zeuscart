Category

ZeusCart provides you privilege to create unlimited number of Categories and Sub Categories. Using Categories module, Admin can create both Category and Sub Category names to be listed. The image for categories can be added. Admin has the facility to edit or delete both category and sub category names. Admin can add landing content directly for both categories and sub categories while creating the names list. The status of categories and sub categories to be displayed in site can be decided by Admin.

 

  1. Click Catalog > Categories > Category will open Main Category List page.
    • The list of all currently added categories will be displayed with information like Category Name, Category Description, Category icon, Status and Options to Edit or Delete option.

 

 

  1. Search: Enter the keyword that you know about a category name on respective text box provided below the corresponding field names and then click Search button will display list of category names based on keyword entered.

 

 

  1. Click on the name of category will open SUB CATEGORY LIST page.

 

Adding New Category

  1. Click button, will open Add Category page.
    1. Category Name: Enter any name for category to be added.
    2. Category Level: This indicates the level of category names to be added. The Parent meant for Category names and Child meant for Sub Category name.
      • If you need to add the category name, then select Parent option and then follow the Steps from iii to viii to add the Category name successful.

 

 

      • If you need to add the sub category name, then select Child option.

 

 

        • Select Parent: The list of all currently added Main Category names will be displayed. Select the category name for which you need to add sub category.
        • Then follow the steps from iii to v.
        • Category Image: Click "Browse" to upload image for the sub category name added. The added image will be displayed in the set Thumbnail Image dimensions.
        • Sub category Special Attributes: The list of all currently added attributes will be displayed. Select the attribute name that you need to add. If you want select the multiple attributes for your sub category, then press CTRL key and select the attributes that you need.
        • Then proceed with the step vi and click Submit button to add the Sub Category name successful.
    1. Category Description: Admin should enter the detailed description of category name added. It helps you to provide full description for the added category.
    2. If any Category Landing Content: When you click on Click Here link, the following two fields will be displayed:
      • Select Landing Content: The list of all currently added Landing Content names will be displayed. Select the content which you need to add for your category.
      • If you need to add new content, then click link which directs you to the CONTENT MANAGEMENT page.
    1. Preview: When a landing name is selected from the list, its corresponding content description will be displayed.
    2. Status: Click ON option, if you want to activate your category name in the site. If you do not want the added category name to be displayed then click OFF option.
    1. Click Submit button, will display "Category Added Successfully" message.

 

Editing Main Category

  1. Click icon will open Edit Main Category page.
    • Update the necessary information to be modified in the fields provided.
    • If you wish to activate the Category then click ON else click Off option.
    • Click Update Main Category will display "Category <name of category> Updated Successfully" message.
    • Click will get you back to Main Category List page.

 

Deleting the Main Category

  1. Click icon will display "Are you sure want to Delete this Category?" message.
    • Click OK will display "Category <name of category> Deleted Successfully" message.
    • Click Cancel to exit from deletion.
