Attribute

Attributes are accessories of a product that it is considered as the special features of product. Admin can add the Attributes ease and effective. We have provided facility to add unlimited number of attributes. Admin has the privilege to update/delete attribute, whenever necessary. Attribute module provides you benefit to add attribute names and its values individually i.e., it involves two separate modules.

 

  • Click Catalog > Attributes > Attribute, will open Attributes page.

      • By default, the page will list the names of all currently added attributes with icons to Edit and Delete in a grid.

     

     

      • Add Attributes: Enter the name of attributes to be added in text box provided. The Special Characters are not allowed between attribute names.
      • Click Add button, will list the added attributes in a grid and it display "Attribute <name of the added attribute> Added Successfully" message.
    1. When you click on button, you will be directed to Attribute Value page where you can add the values for the entered attributes.
    2. Click icon, will open Edit Attributes page.
      • Attribute: Displays the selected attribute name and if you need to make any change, update the name of attribute.
      • Click Update button, will display "Updated <name of the updated attribute> Successfully" message.
    1. Click icon, will display "Are you sure want to Delete?" message.
      • Click OK will display "Deleted Successfully" message.
      • Click Cancel to skip from deletion.
