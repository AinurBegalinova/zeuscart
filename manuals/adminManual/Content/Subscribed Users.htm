Subscribed Users

This module helps Admin to view all the mail id of users who have subscribed for Newsletters. For easy use of Admin, this module provides facility to create the newsletter and view all the created newsletters from this module itself.

 

  1. Click Promotions > Newsletter > Subscribed Users will open Newsletter Subscription List page.

 

 

  • The list of all the users who have subscribed for newsletters while registration and without registration will be displayed.
  1. Click View Newsletter link will open NEWSLETTER LIST page.
  • The list of all the created newsletters currently will be displayed with title name of newsletter, the date in which it was created, status of newsletter and option to delete it.
  1. Click button will open NEWSLETTER page.

We have provided Paging for viewing the subscribed user details. Click on the page numbers to go to corresponding pages and click Prev and Next to go back and to go next to the page where you are now. By default Ten subscribed users will be listed in a page.
