Html Contents

ZeusCart provides you the separate module to manage landing contents added. Admin can add any number of landing contents and make it display anywhere in website using the Category module. Admin has the privilege to edit/delete the contents, whenever necessary.

 

  1. Click CMS > HTML Contents will open Show Contents page.
    • The list of entire currently added contents will be displayed with the name of content and options to edit/delete.

 

 

  1. If you want create a new content, then click link will open Content Management page.

 

 

    • HTML Content Name: Enter the name for your landing content to be displayed.
    • HTML Content: Enter the description of landing content name added by you.
    • Click Add Content button, will display "Content Added Successfully" message.
  1. If you wish to update landing content, click link, will open Content Management page.
    • Update the necessary information needs to be changed.
    • Click Update Content will display "Contents Updated Successfully" message.
  1. If you need to delete any content, click link provided corresponding to the content information, will display "Are you sure want to Delete?" message.
    • Click OK will display "Content Deleted Successfully" message.
    • Click Cancel will exit from deletion.

We have provided Paging for viewing the HTML Contents added. Click on the page numbers to go to corresponding pages and click Prev and Next to go back and to go next to the page where you are now. By default Ten HTML Contents will be listed in a page.
