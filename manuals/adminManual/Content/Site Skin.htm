Site Skin

Using different types of Skins for a site seems every other day, there is a new Website Builder on the market. The truth is they are catching on. So by default, ZeusCart provides you four different skins for your website. Moreover, ZeusCart gives you the facility to add your skin style to be displayed in site.

 

  1. Click Settings > Site Settings > Site Skin will open Available site Skin page.

 

 

  1. By default, it displays four different skins to select your skin to be displayed. Select the Skin color you wish to add.
  2. Click Update Site Skin will display "Site skin is updated to <selected skin name> Successfully" message.

 

Adding your own skin

  • Click button provided in the Available site Skin page, then it will open Add Skin page.
    • Skin Name: Enter the name of your skin to be displayed.
    • Css Zip File: Click "Browse" to add zip file of your skin.

 

 

Steps to create your own skin

      1. Go to root folder in FTP and open css folder.
      2. In css folder, you can find folders with the names of default skins provided with ZeusCart.
      3. If you like to edit the content of skins provided, then copy any of the skin provided and save it in any folder of your system.

For Example: If you have copied "gray" folder, you can find images folder and style.css file available in the "gray" folder.

Remember you should not change the name of both images folder and style.css file.

      1. You can replace your own skin images in the images folder available in "gray" folder.
      2. In style.css file, you should not change the style class names provided but you can change the values of style classes.
      3. After all your modifications got completed, then zip both images folder and style.css file and upload it in "Css Zip File" field.

Do not copy both image folder and style.css file in a separate folder to make zip of it. It will collapse your skin. So just simply do zip of both image folder and style.css file and upload it.

  • Click Submit will add your own skin with the available skins.
