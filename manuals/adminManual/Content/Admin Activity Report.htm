Admin Activity Report

Admin Activity Report module helps Admin to know the details of activities like opening pages in Admin Console. It displays the list of details like name of persons who logged in to site and the pages he opens in site. The date and time in which the page has opened and each and every details will be displayed. This makes Admin to know the site activity easily. So that Admin can monitor the Sub Admins activities to maintain the site properly.

 

  1. Click Others > Admin Activity Report will open Admin Activity Report page.
  2. The list of details like User name, URL which the user has opened in site and the visited details like, day, date and time of opening the page.

 

 

We have provided Paging for viewing the user details. Click on the page numbers to go to corresponding pages and click Prev and Next to go back and to go next to the page where you are now. By default 20 customers will be listed in a page.
