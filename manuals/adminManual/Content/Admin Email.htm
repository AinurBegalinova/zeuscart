Admin Email

This module helps Admin to add email id to be used for all transactions. The added email id here will be used to send mails to the end users at any time.

 

  1. Click Settings > Site Settings > Admin Email will open Administrator Email Settings page.

 

 

  1. Administrator Email: Enter the email id of Admin to be used.
  2. Click Update email button will display "Site Email Settings <name of entered email id> Saved Successfully" message.
