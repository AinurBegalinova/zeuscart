Attribute Value

ZeusCart 2.3 provides you support to add unlimited number of attribute values to a product. Admin can add the Attributes easily. We have provided facility to add unlimited number of attribute values. Admin has the privilege to update/delete attribute values, whenever necessary.

 

  1. Click Catalog > Attributes > Attribute Value, will open Attribute Values page.
    • By default, the page will list the names of all currently added attributes and their respective attribute values with the icons to Edit and Delete in a grid.

 

 

    • Add Attribute Values: From the list box, select any attribute name for which you need to add attribute value. Add the attribute value in the text box provided.
    • Click Add button will display "Attribute Values <name of the added attribute value> Added Successfully" message.
  1. When you click on button, you will be directed to Attribute page where you can add the attributes.
  2. Click icon, will open Edit Attribute Values page.

 

 

    • Attribute Values: Displays the selected attribute value name and if you need to make any change, update the value of attribute.
    • Click Update button, will display "Attribute Value <name of the updated attribute value> Updated Successfully" message.
    • Click button will get you to Attribute Values page.
  1. Click icon, will display "Are you sure want to Delete?" message.
    • Click OK will display "Deleted Successfully" message.
    • Click Cancel to cancel the deletion.

When you delete the attribute value, only the value of attribute will be deleted not the name of attribute.

We have provided Paging for viewing the attributes details. Click on the page numbers to go to corresponding pages and click Prev and Next to go back and to go next to the page where you are now. By default Ten attributes will be listed in a page.
