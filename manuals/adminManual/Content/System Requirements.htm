System Requirements

At the base level, ZeusCart 2.2 will require the following software:

 

  • Supported Operating Systems:
    • Linux x86, x86-64 and Windows.
  • Supported Web Servers:
    • Apache 1.3.x.
    • Apache 2.0.x .
    • Apache 2.2.x.
  • Supported Browsers:
    • Microsoft Internet Explorer 6 and above.
    • Mozilla Firefox 3.0.
  • PHP Compatibility:
    • 5.2.2.
    • Safe_mode off.
    • Memory_limit 32M or more.
  • MySQL:
    • 5.0.
    • MyISAM storage engine.
  • SSL:
    • If HTTPS is used to work in the admin, SSL certificate should be valid. Self-signed SSL certificates are not supported.
