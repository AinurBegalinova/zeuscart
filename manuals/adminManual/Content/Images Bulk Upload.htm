Images Bulk Upload

ZeusCart 2.3 provides you an advanced feature of uploading the images for products in bulk. This facilitates Admin for saving the time in uploading the images for products. To upload the images successful, Admin has to follow the steps given below.

 

Before downloading the sample file from Upload The Images TSV File page, you must follow the below steps:

  1. Open your FTP.
  2. Open Images folder available in root folder.
  3. In the images folder, create a new folder with the name upload images.
  4. In the upload images, add all product's images that are going to be uploaded in bulk. You can add both the thumbnail images and also the main images.

Then you open ZeusCart 2.3 Admin Console,

  • Click Catalog > Products > Images Bulk Upload will open Upload The Images TSV File page.
  • Sample Images TSV File: Click icon will open a dialog box to download the sample Zip file.

 

 

  1. After downloading Sample file from ZeusCart 2.3, you will be seeing the two files in Zip folder named productids and image.
  2. In productids file you can view the list of product details you have added in site with its Product id, Title, brand, Category name, Sub Category name, Introduction date, Maximum Selling Retail Price (MSRP), Featured Item Flag.
  3. In Image file you can view the field names such as product id, image path and image type.

You should not make any corrections in the field names and you can enter the product details based on field names provided.

  1. In the downloaded two files Image file acts as a main file for uploading the Images in bulk,
    1. In Image file, you can see the field names like product id, image path and image type.
    2. product id: Enter the id of product. You can collect the product id from product ids file that you have downloaded.
    3. image path: Enter the path in which you have added the images in FTP.

You should enter the valid path. Suppose if you have uploaded the image with the name sample.jpg in your root > images > upload images folder. Then you must enter the image path as root/images/uploaded images/sample.jpg.

    1. image type: In this field, if your image type is Thumbnail then enter as thumb. If your image type is main then enter as main.
    2. After completing the image details save the image file.
  1. Images TSV File: Click Browse button to upload the TSV file in which you have completed your all image entries.
  • Click Submit will display "Image(s) Uploaded Successfully" message.
