What's New?

Below is a list of the features that make up ZeusCart 2.3, some of which are new and exclusive to this version. The new features for ZeusCart 2.3 are highlighted in green.

 

General Features

  • Cross platform compatible (Windows, Linux).
  • Automatic image resizing of products & categories.
  • Developed in MVC pattern using AJDF standard.
  • Developed in object oriented approach using PHP 5
  • New template integrated.
  • Site Installer.

Administrator Console

  • Price comparison (using pricerunner.com API).
  • Customizable Category landing content.
  • Customizable sub Header.
  • Sub admin.
  • Role based sub admin.
  • Google Analytics enabled
  • Google Base data feed support. (listing your products in www.google.com/products in).
  • N numbers of extended product specifications (attribute => Screen Size , attribute value=> 12”, 14” ,21” …).
  • Newsletter subscription management.
  • Customizable Time Zone.
  • Statistics for products, customers, orders and categories.
  • Support for homepage banner.
  • User data export feature to Excel, CSV,TAB and XML formats
  • Google AdWords Text Ad creation.
  • Coupon creation at admin.
  • Product data bulk upload.
  • Images bulk upload.
  • Category bulk upload.
  • Order shipment tracker at admin & store front.
  • Multiple images for a Product.
  • Most search keywords at admin.
  • Admin activity report.
  • Order data export.
  • Product data export.
  • Category data export.
  • Admin activity export.
  • Creating order from admin panel.
  • Creating user account from admin panel.
  • Site News management.
  • Regional based taxation system.
  • Business intelligence.
  • Simple help tips wherever required.
  • Enhanced listing style at admin.
  • Best selling products in admin.
  • New payment gateways Integrated - Paymate Express, Money Bookers, PSi Gate, StormPay, AlertPay, Your Pay, iTransact, BluePay, SafePay.

Store Front

  • Product comparison.
  • Product comparison snapshot.
  • Narrow search for products (search by product attributes).
  • Discount on MSRP by quantity (Tier price).
  • SEO friendly product pages (use meta contents for each products).
  • Bookmark the product detail page (for all browsers).
  • Featured items (in Homepage, Main category landing page, sub category landing page).
  • Cross selling of products.
  • Permanent shopping cart for registered users.
  • Wishlist.
  • Wishlist snapshot.
  • Move wishlist products to shopping cart.
  • New arrivals list.
  • Recently viewed products.
  • Custom pages (static and dynamic CMS).
  • Add on skins.
  • Html contents management.
  • RSS subscription (for new products).
  • Google AdSense banner (468x60) support.
  • Quick checkout.
  • Product out of stock indication at store front.
  • Moderated product reviews and ratings.
  • Thumbnails and Large Image Views for products.
  • Breadcrumb for key pages.
  • Tag clouds.
  • Explicit Tagged search.
  • Narrow Search redefined.
  • My product reviews.
  • CMS pages links made like menu.
  • Best selling items at store front.
  • My Account page.
  • Guest checkout.
  • Redeeming coupon when checkout.
  • Product quick view.
  • More new products page.
  • More Featured products page.
  • Collaborative Filtering - “Who Bought This Item Also Bought this products”.
  • Order quick view.
  • Cash on Delivery (admin & store front ) 67.Pay in Store.
  • Site Hit counter.
  • Address book management for users.
  • Select address from address book when checkout.
  • Latest news at store front.
  • Captcha code for form inputs.
  • Move products from cart to wish list while checkout.
