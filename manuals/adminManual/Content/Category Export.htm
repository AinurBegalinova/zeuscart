Category Export

ZeusCart 2.3 provides you separate module to export the category details. Admin can export the category details in four formats like Excel, XML, CSV and Tab file.

 

  • Click Catalog > Categories > Category Export will open Export category page.

 

 

  • Click on the respective format you need to export the report.
  • Click Export button, then you can download the file.

Suppose if you need to export the category details in Excel format, and then select Excel from the list provided, then click "Export" button. It will open "Opening category_Details.xls" dialog box, Click "OK" will download excel file otherwise click "Cancel" to exit from downloading the file.
