Tax Settings

ZeusCart 2.3 comes with the advanced feature of applying tax amount for your products. We have provided easy functionality in setting the tax values. Admin can add apply tax to products based on two different ways such as apply to specific countries and apply to all products. Admin can add number of tax values. Admin can create the tax value based on the tax to applied for billing address or shipping address, subtotal amount of product or sub total + shipping amount of product and much more.

 

  1. Click Settings > Tax Settings will open Tax Settings page.
  2. I don't want to apply tax to any products in my store: When you select this option then there will not be tax value for any of your products.
  3. I want to apply tax for specific countries and/or states: If you wish to use the tax value for specific countries and state, then you can select this option. When you select this option, then you must follow the below steps:
  • Click button.

 

 

  • Tax Name: Enter the name of your tax. You can assign any name for your tax which you are going to create.
  • Apply Tax to: Select the options from the list to specify for which you are going to apply this tax. The selected option will be applied when user checkouts for the products.
  1. If you select Subtotal, then the tax will be applied for the sub total amount of the products.
  2. If you select Subtotal+Shipping, then the tax will be applied to both sub total amount and shipping amount of the products.
  • Apply To: Select the name of country for which you are going to apply this tax. When a country is selected obviously the tax value will get apply to its corresponding state names.
  • Charge tax on: Decide whether you are going to apply this tax based on the billing address or shipping address of the products which user is checking out.
  • Tax Rate: Enter the value of tax which is calculated in percentage.
  • Status: Enable the checkbox to activate the tax.
  • Click Add Tax Rate will display "Tax Rate Added Successfully" message.
  • Click icon to update the tax added.

 

 

  • Update the necessary information.
  • Click Add Tax Rate will display "Tax Rate Modified Successfully" message.
  • Click icon will display "Do you want to delete" message.
  • Click OK will display "Tax Rate Deleted Successfully" message.
  • Click Cancel to exit from deletion.
  1. I want to apply one tax rate to all products in my store: If you wish to use the tax value for all products, then you can select this option. When you select this option, then you must follow the below steps:

 

 

  • Tax Rate Name: Enter the name of your tax. You can assign any name for your tax which you are going to create.
  • Based on: Select the options from the list to specify for which you are going to apply this tax. The selected option will be applied when user checkouts for the products.
  • If you select Subtotal, then the tax will be applied for the sub total amount of the products.
  • If you select Subtotal+Shipping, then the tax will be applied to both sub total amount and shipping amount of the products.
  • Tax Rate: Enter the value of tax which is calculated in percentage.
  • Click Update Tax Setting will display "Tax Settings Changed Successfully" message.
