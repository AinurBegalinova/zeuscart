Featured Products

ZeusCart 2.3 is providing you allocation for listing the featured products. Admin can add the featured products simple and fast using Select Featured Products module. This helps users to know the featured products ease. The selected featured products will be displayed in home page of site.

 

  1. Click Catalog > Products > Featured Products will open Select Featured Products page.
    • Main Category: Select a Category Name to be added as featured products from the list box provided.
    • Sub Category: The list of Sub Category names will be displayed based on selected Category name. Select the Sub Category name to be added.
  1. The list of all Products available in selected Category and Sub Category names will be displayed.

 

 

  1. Select the product that needs to be added as featured product by enabling the check box provided under the Featured field.
  2. Click Add will display "Featured Products Added Successfully" message.
