Order List

Order List helps Admin to track all the orders made by users. Search engine helps Admin to search for orders easy with any of keywords from name of product to date in which it was ordered. When a product is just got ordered from site, then it will be displayed as Pending status. When Admin viewed that order and processed for shipping then can change the status from Pending to Processing. After the product got delivered to user, then Admin can change the status of order from Processing to Delivered.

 

Step 1: Click Orders > Order List will open Order Management page.

  • The list of ordered product information will be displayed in a grid with the name of product, id of order, date in which order has been made, name of bill, name of the shipper, amount of order and also the status.

 

 

Step 2: Search: Using search, Admin can find the orders easily and it helps in saving your time.

  • Enter the keyword that you know about product order in the respective text box provided corresponding to field names.
  • Click Search button will display the list of order information based on entered keyword.

Step 3: Viewing the Order Details - When you click on the + sign near the Order Id, you can view complete details of order like Billing Address, Shipping Address, Ordered Product's Information below.

Step 4: Changing the Status of Order - If you need to change the status for a product.

  • Select the check box available corresponding to the product order.
  • Click on the Status (Pending/Processing/Delivered) link provided will open ORDER IN DETAIL page.

 

 

  • The order details will be displayed with the Order ID, Customer Name, Order Date, Close Date and much more.
  • Order Status: Select the order status from the list box available. you can change from pending to delivered, delivered to pending and so on.
  • Shipment Name & Track Id: Select the Shipment Name from the list box and enter the id using which you have delivered the product.

The entered Track Id will be displayed for user to know the status of shipping from the corresponding shipment website.

  • Click Submit will display "Updated Successfully" message.
  • Click Back To Order List link will get you back to Order Management page.

We have provided Paging for viewing the order details. Click on the page numbers to go to corresponding pages and click Prev and Next to go back and to go next to the page where you are now. By default Ten order details will be listed in a page.
