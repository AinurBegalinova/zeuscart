Promotions | Newsletter | View

 

View

Newsletter is the module used to create and send the newsletter who have subscribed for newsletters while registering with ZeusCart. Admin can create any number of Newsletters and send them to subscribed users automatically in single click.

 

  1. Click Promotions > News Letter > View will open Newsletter List page.
    • The list of all the created newsletters currently will be displayed with title name of newsletter, the date in which it was created, status of newsletter and option to delete it.

 

 

  1. If you want to create a new Newsletter click button will open NEWSLETTER page.

 

 

    • Newsletter Name: Enter the name or title for your newsletter to be created.
    • Newsletter Content: Enter the description or content for Newsletter and make an attractive newsletter using WYSIWYG editor.
    • Click Add Newsletter will display "Newsletter <name of the newsletter> Created successfully" message and will add the created newsletter in Newsletter List page.
    • Click View All Newsletters link will get you to Newsletter List  page.
  1. If you wish to change any information in created Newsletters, click on the "name of newsletter" added will direct you to EDIT NEWSLETTER page.
    • Update the necessary information needed.
    • Click Update Newsletter button will display "Newsletter Updated successfully" message and from there you can send the Newsletter using the Send Newsletter button.
    • Click Send Newsletter button will send the newsletter to all subscribed users and will display "Newsletter has been sent successfully" message. Then the status of corresponding newsletter will be changed in to Sent in the Newsletter List page.
    • Click will direct you to create a newsletter in Newsletter page.
  2. If you like to delete any Newsletter, then click icon, will display "Are you sure want to Delete?" message.
    • Click OK will display "News Letter Deleted Successfully" message.
    • Click Cancel will help you to skip from deleting the newsletter.
