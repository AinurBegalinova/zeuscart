Currency Settings

Admin can set multiple currencies in ZeusCart 2.3. This is a new feature added in version. Admin can add the multiple currencies. While viewing a product price, users can view it any currencies in which he needed. When user just clicks on the country name, the product price will be displayed based on that particular country. Admin can add any number currency values and has the privilege to update/delete the currency values, whenever needed.

Remember multi currency can be used only to see the conversion price for the country. But user can checkout the product only with the primary currency selected Admin.

 

  1. Click Settings > Currency Settings will open List of Currency page.
  • The list of currencies that are added currently will be displayed with the information like Currency Name, Currency Code, Conversion Rate, Applied To and the Status. By default, the primary currency created while installation will displayed which can be updated but can not be deleted.
  1. Click button will open Currency Settings page.

 

 

  • Currency Name: Enter the primary currency you wish to use for your entire site. Based on the currency you have selected only all the transactions will be processed.
  • Currency Token: Enter the token of your currency. You can consider it as the symbol of currency.
  • Country: Enter the country name of your currency
  • Currency Code: Select the code of currency name that you have you entered from the list box.
  • Conversion Rate: Enter the conversion rate of your currency with respective of US dollar.
  • Status: Enable the check box to activate the currency in store front.
  • Click Add Currency will display "Added Successfully" message.
  1. Click icon will open Currency Settings page.

 

 

  • Update the necessary information in the provided text box.
  • Click Add Currency will display "Updated Successfully" message.
  1. Click icon will display "Do you want to delete" message.
  • Click OK will display "Deleted Successfully" message.
  • Click Cancel to exit from deletion.
