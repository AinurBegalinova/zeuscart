Log On

To Sign in to ZeusCart 2.3 Admin Site:

 

Step 1: Click on your Internet icon. You need Firefox/Microsoft Internet Explorer.

Step 2: Type in your domain address for ex: http://www.yourdomain.com/admin (you may wish to save this address to your favorites so you do not have to type the address in again).

 

 

Step 3: In the provided User Name and Password fields, type in your User Name, press TAB to move your cursor to the Passwordfield, then type in your password.

Step 4: Press ENTER or click on the Login button.

 

Forgot Password

Forgot Password method helps you to get the admin password. When Admin enters the mail id which was given at installation time then password will be sent to that email id automatically.

 

Step 1: If you type in the wrong Login ID and/or Password, click "Forgot Password" link.

 

 

    • Enter the Email id you have given at the installation time
    • Click "Send My Password" button will display "User name and Password has been sent to <entered mail id name> successfully" message.
    • Click "Back to Login" link will direct you to the Login page.
