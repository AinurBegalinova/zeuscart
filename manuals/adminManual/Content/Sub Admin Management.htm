Sub Admin Management

Sub Admins are the administrators created by Main Admin to maintain and control the site and also to provide a booming site to the users. ZeusCart supports you the feature to add any number of sub admin. Admin has the privilege to view profile of all Admin Users and also to edit/delete Admin Users, whenever needed. Admin has the facility to make an Admin User enable/disable. Admin can allot different roles for Sub Admin to analyze the site. Except the roles allotted by Admin, the specific sub admin cannot do any of the other roles. But Admin can allot the same role to different sub admin. This makes Admin to give the best E-Commerce site to users.

 

  1. Click Sub Admins > Sub Admin Management will open Sub Admin Management page.
    • This list of all currently added sub admin information will be displayed in a grid with their Names, Password, Mail id, Status and Options to Edit/Delete and their role maintenance.
  1. Adding Sub Admin

The grid will consist of field names in the order of Sub Admin Name, Password, Sub Admin Mail and Status. If you want to add a new Sub Admin, enter the details of Sub Admin in the provided text box of corresponding fields. Enable the Check box provided under the Status field to activate the Sub Admin in site otherwise leave the check box blank.

 

 

    • Sub Admin Name: Enter the name of Sub Admin to be added. No Special characters are allowed to enter Sub Admin names.
    • Password: Enter the Password to be used for corresponding sub admin name added. The characters allowed to enter the password are minimum 6 to maximum 15 characters.
    • Sub Admin Mail: Enter the mail id of sub admin added.
    • Status: Enable the Check box provided, if you need to activate added sub admin in site.
    • Click Add Sub Admin button will add sub admin and will make it display in the grid.
  1. Steps to edit Sub Admin information
    • Click icon provided corresponding to the Sub Admin details to be updated, will open EDIT SUB ADMIN MANAGEMENT page.
    • Update the necessary fields to be changed.
    • Click "Update" button, will add changes to the respective sub admin.
  1. Deleting Sub Admin
    • Click icon available respective of the sub admin to be deleted, will display "Do you want to delete" message.
      • Click OK will delete the sub admin from site.
      • Click Cancel will help you to skip from deletion.
  1. Allotting Roles to SubAdmin
    • Click Roles link will open SUB ADMIN ROLE MANAGEMENT page.

 

 

      • The list of role names will be displayed with the check boxes near by to select the roles.
      • Click the Check Boxes provided respective of the roles to allot that roles to the selected sub admin.
      • Click Update button will add selected role to the corresponding sub admin and will display "Updated Successfully" message.
    • If you wish to remove the allotted roles to sub admin, then just uncheck the check boxes displayed.
    • When all your actions are completed, Click Back link to go back to Sub Admin Management page.

We have provided Paging for viewing the sub Admin details. Click on the page numbers to go to corresponding pages and click Prev and Next to go back and to go next to the page where you are now. By default Ten Sub Admins will be listed in a page.
