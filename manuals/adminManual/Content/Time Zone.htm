Time Zone

Use of Time Zones helps Admin to select and display time, based on the selected country. Admin do not need to set the time based on country to be used. When just he selects the country name displayed in list box will display the right time in site based on country selection. This makes Admin ease use of time zone and save his time in setting up of right time.

 

  1. Click Settings > Site Settings > Time Zone will open Time Zone page.

 

 

  1. Current Time Zone: It will display the current selection of time zone used in site.
  2. Set Time Zone to: Select the name of country to display the corresponding right time in site.
  3. Click Set Time Zone button will display "Timezone Updated to <name of the selected time zone> Successfully" message.
